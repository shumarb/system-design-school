# System Design School

## Overview
This repository is a summary of concepts learnt on [System Design School](https://systemdesignschool.io/).

## Introduction
System Design is the process of `architecting` & `planning` applications by carefully selecting various data to efficiently handle data processing & storage needs.
This involves:
- Understanding the requirements of the application.
- Identifying the right data systems to manage diverse tasks.
- Using application code to work together in harmony.
